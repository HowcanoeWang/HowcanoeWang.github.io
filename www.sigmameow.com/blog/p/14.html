2. 环境搭建

Python3.6(64位)+Pycharm Community安装过程略

2.1 基础库的下载与安装


https://www.lfd.uci.edu/~gohlke/pythonlibs，记得找对应的Python版本(3.6)和对应的系统位数(32位没有_amd64.whl)

  • numpy‑1.15.4+mkl‑cp36‑cp36m‑win_amd64.whl
  • PyQt4‑4.11.4‑cp36‑cp36m‑win_amd64.whl
  • VTK‑8.1.2‑cp36‑cp36m‑win_amd64.whl
  • mayavi‑4.6.2+vtk81‑cp36‑cp36m‑win_amd64.whl
  • traits‑4.6.0‑cp36‑cp36m‑win_amd64.whl
  • traitsui‑6.0.0‑py3‑none‑any.whl

建议先建立虚拟环境(我建立一个一个py3d的虚拟环境)，创建虚拟环境的基础操作可以百度或者参考这篇文章

使用cmd进入虚拟环境之后，请按照上面的顺序进行安装(后面的包依赖前面的包) 

(py3d) D:\...\Python\py3d>pip install 下载地址\xxx.whl

【如果安装的过程中报错】
如果电脑第一次安装python，并且之前没有从事过编程的活动的话，Windows系统上很有可能提示没有安装Microsoft Visual C++ compiler，可以参考下面的连接下载安装对应的VS版本

https://wiki.python.org/moin/WindowsCompilers

由于我们用的是python3.6，所以下载Visual C++ 14.0 版本即可；详细的安装教程见连接

TVTK帮助资源

http://www.vtk.org/doc/nightly/html/annotated.html
看的过程中注意，上面的文档是VTK的文档，在转成python库的过程中，变量名进行了以下的变更：

  • TVTK库中的类名去除了前缀vtk
  • 函数名按照Python的惯例，采用下划线连接单词
    > AddItem -> add_item
  • VTK对象的方法在TVTK中用Trait属性替代
    如在VTK中m.SetInputConnection(c.GetOutputPort())变成了m.input_connection(c.output_port)
